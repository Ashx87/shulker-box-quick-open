# Shulker Box Quick Open

A simple Fabric mod that allows you to quickly open shulker boxes directly from your hand without placing them down.

## âœ¨ Features

- **Quick Access**: Right-click while holding a shulker box and looking at the sky/air to instantly open it
- **Full Functionality**: Uses the vanilla shulker box interface - all original behaviors are preserved
- **Safe**: Prevents the opened shulker box from being placed inside itself (which would cause item loss)
- **Works on Servers**: Fully compatible with multiplayer servers

## ðŸ“– How to Use

1. Hold a shulker box in your **main hand**
2. Look at the **sky or air** (not at any block)
3. **Right-click**
4. The shulker box opens instantly!
